# Пользовательский Тетрис (Unity 6)

Мобильный Тетрис, разработанный на Unity 6 для Android.

## Особенности

- Игровой процесс Tetris на основе сетки
- Настраиваемые фигуры
- Архитектура на основе JSON
- Сенсорное управление на мобильных устройствах
- Система подсчета очков
- Система уровней
- Предварительный просмотр следующей фигуры
- Система «Игра окончена»
- Жесткое падение
- Очистка линий

---

# Платформа

- Android
- Портретная ориентация

---

# Управление

## ПК
- Левая стрелка → Перемещение влево
- Правая стрелка → Перемещение вправо
- Стрелка вверх → Вращение
- Стрелка вниз → Мягкое падение
- Пробел → Жесткое падение

## Мобильные устройства
- Свайп влево/вправо → Перемещение
- Свайп вниз → Жесткое падение
- Касание → Вращение

---

# Архитектура

Структура проекта:

- GridManager → управляет сеткой игрового поля
- Piece → перемещение и вращение фигуры
- PieceSpawner → создает фигуры
- ConfigLoader → Загружает фигуры из JSON
- UIManager → Логика пользовательского интерфейса
- ScoreManager → Система очков и уровней

В игре не используется физика Unity.
Все столкновения и перемещения рассчитываются математически в координатах сетки.

# Настраиваемые фигуры

Фигуры загружаются из конфигурации JSON.

Пример:

```json
{
"id": "T-shape",
"color": "#800080",
"pivot": { "x": 1, "y": 1 },
"blocks": [
{ "x": 0, "y": 1 },

{ "x": 1, "y": 1 },

{ "x": 2, "y": 1 },

{ "x": 1, "y": 0 }

]
}
